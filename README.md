# nlwheat
Rocketseat Out-2021

Aplicação criada durante a participação do NLW Heat (RocketSeat).

***** BACKEND *****
--------------------------------------------
1. Instalação
cd backend [enter] 
yarn [enter]

2. Criar .env (com base no .env.example)

3. Criar o Banco de Dados PRISMA
yarn prisma migrate reset [enter]

4. Executar
yarn dev [enter]
--------------------------------------------

***** FRONTENDWEB *****
--------------------------------------------
1. Instalação
cd frontendweb [enter]
yarn [enter]

2. Executar
yarn dev [enter]
--------------------------------------------
